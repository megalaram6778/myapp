# CI/CD Pipeline using Jenkins and Docker

## Project Overview
This project demonstrates a CI/CD pipeline using Jenkins and Docker.

## Tools Used
- Jenkins
- Docker
- Git
- GitHub
- Python

## Project Files
- Dockerfile
- Jenkinsfile
- requirements.txt

## Workflow
1. Clone the repository.
2. Jenkins detects changes.
3. Build a Docker image.
4. Run the application inside a Docker container.
